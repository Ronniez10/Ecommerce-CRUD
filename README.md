PRODUCT API
-------------

http://localhost:8080/api/products/{productId}

http://localhost:8080/api/all/

http://localhost:8080/api/products/

PUT Request for a Product

http://localhost:8080/api/products/

DELETE A Product

http://localhost:8080/api/products/{productId}
